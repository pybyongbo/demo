svg初探!

SVG是神马？

  • SVG指可伸缩矢量图
  • 用来定义基于矢量的网络图形
  • 使用XML格式来定义
  • 在放大或缩小的情况下，图形质量不受影响
  • 在2013年成为W3C推荐标准
  • 浏览器支持情况：IE9+，chrome 33.0+，Firefox 28.0+，Safari 7.0+

SVG使用方式？

使用方式 代码
用 iframe标签嵌入SVG <iframe src="xx.svg" width="200" height="200" > </iframe>(svg标签)
用 img 嵌入SVG < img src="xx.svg" width="300" />
用CSS背景图的方式嵌入SVG div { background: url('xx.svg') no-repeat center; background-size : 200px 200px; }
直接使用 svg标签 <svg width="600px" height="300px" viewBox="0 0 250 250"> </svg>

eg:直接在html中使用svg标签.

.svg文件,常见写法是需要xml申明的,如果是直接在HTML中使用svg的话,就不用在写xml申明了...

SVG图形、属性及基本操作

基本图形

标签名 标签代码
矩形 <rect>
圆型 <circle>
椭圆 <ellipse>
线 <line>
线条 <polyline>
多边形 <polygon>
路径 <path>(后面有时间重点介绍.)
组 <g> <defs>
图片 <image>
描述 <title> <desc>
文本 <text>

基本属性

属性名称 含义
fill 填充颜色（rgb 值、颜色名或者十六进制值）
stroke 定义图形边框颜色
stroke-width 定义图形边框的宽度
transform 声明自身坐标相对父级坐标系的变换，很重要的属性，后面会深入探讨

基本操作

操作名称 方法
创建图形 document.createElementNS(ns, tagName) ns是svg的命名空间
添加图形 element.appendChild(childElement)
设置/获取属性 element.setAttrbute(name, value) element.getAttribute(name)

创建的代码

var SVG_NS = 'http://www.w3.org/2000/svg';
var svg = document.createElementNS(SVG_NS, 'svg');
svg.setAttribute('width', '100%');
svg.setAttribute('height', '100%');
document.getElementById('svg-wrap').appendChild(svg);
